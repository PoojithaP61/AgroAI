================================================================================
  COMMON ERRORS & FIXES:
================================================================================

ERROR: "ModuleNotFoundError: No module named 'fastapi'"
FIX: Make sure you activated virtual environment (Step 3)
     You should see (finalenv) at start of your prompt

ERROR: "No module named 'backend'"
FIX: Make sure you're in the project root directory (Step 2)
     Run: cd C:\Users\pooji\OneDrive\Desktop\AgroAI_FinalYear_Project

ERROR: "Cannot activate virtual environment"
FIX: Try running PowerShell as Administrator
     Or recreate venv: python -m venv finalenv

ERROR: "Port 8000 already in use"
FIX: Kill the process using port 8000, or change port in backend/start.py

ERROR: "ML models failed to initialize"
FIX: This is OK! Server will still run, but prediction endpoints won't work
     Make sure ml/encoder/encoder_supcon.pth exists
